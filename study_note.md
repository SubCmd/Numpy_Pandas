# 학습한 개념 정리

1. 윈저라이징 (winsorizing)
: 데이터 분석에서 극단적인 이상치(outlier)의 영향을 줄이기 위해, 가장 작은 값과 큰 값을 특정 백분위수의 값으로 대체하는 데이터 처리 기법
-> outlier를 제거하지 않고, 경계값으로 눌러버리는 기법
    - 장점 : 데이터 손실율이 적고, 이상치로 인한 평균 및 표준편차 왜곡 완화
    - 활용 : 매출, 유저 행동 데이터 등 극단값이 많은 경우, A/B 테스트에서 variance 줄이고 싶을 때,

2. 버킷팅 (bucketing)
: 연속형 데이터를 구간(카테고리)으로 나누는 것
